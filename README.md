## 🚀 Proyectos Destacados

### 🌍 [Sistema de Analítica - Mundial 2026](https://github.com/miguelASL/futbol_prediccion) ⭐
> Proyecto principal: pipeline end-to-end para predicción deportiva con ML

- ETL: Transfermarkt scraping → Python → SQL Server.
- ML: XGBoost + LightGBM + RandomForest (68%+ accuracy).
- Dashboard: Power BI con análisis de value bets.
- Alertas: reportes automáticos y sistema de trading.

**Tecnologías:** `Python` `SQL Server` `Power BI` `XGBoost` `LightGBM` `Transfermarkt API` `Power Automate` `Plotly`
