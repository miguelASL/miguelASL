## 🚀 Proyectos Destacados

### 🌍 [Sistema de Analítica – Mundial 2026](https://github.com/miguelASL/futbol_prediccion)
> *El proyecto más ambicioso del perfil*

Pipeline end-to-end para análisis y predicción del Mundial 2026. Scraping de datos desde Transfermarkt, ETL con Python hacia SQL Server, modelo de predicción con Ensemble Learning, y dashboard interactivo en Power BI.

`Python` `SQL Server` `Power BI` `XGBoost` `LightGBM` `Transfermarkt API` `Power Automate`

- 🤖 ML con **68%+ de accuracy** (XGBoost + LightGBM + RandomForest)
- 📊 Dashboard interactivo con análisis de value bets y alertas automáticas
- 🔄 Pipeline ETL automatizado con scripts batch y Python
